Astromesh — Agent Runtime Platform

This product includes software developed by the Astromesh project and its contributors.

Astromesh is an open source platform designed to build, orchestrate, and operate
AI agents at scale. It provides a runtime environment where agents can interact
with multiple models, tools, memory systems, retrieval pipelines, and external
communication channels through a declarative configuration model.

Project repository:
[https://github.com/](https://github.com/)[/astromesh](https://github.com/)
